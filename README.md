# NotifyHub

A distributed notification platform built with ASP.NET Core and Clean Architecture.

## Tech Stack

- ASP.NET Core 8
- Entity Framework Core
- SQL Server
- SignalR
- RabbitMQ
- Redis
- Docker
- xUnit

## Features

- Email notifications
- In-app notifications
- Real-time SignalR updates
- Retry mechanism
- Dead-letter queues
- Caching
